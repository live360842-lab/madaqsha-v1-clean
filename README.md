# مداقشة - نسخة ويب أولى

هذه نسخة أولى قابلة للتشغيل من لعبة **مداقشة** كـ Web App / PWA موجهة للجوال أولًا.

## التشغيل المحلي

```bash
npm install
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## النشر على Railway

1. ارفع الملفات إلى GitHub.
2. اربط المستودع في Railway.
3. Railway سيستخدم `node server.js` من `railway.json` أو `Procfile`.

## ما تم تنفيذه في V1

- شاشة دخول عربية RTL.
- دخول محلي للتجربة.
- زر Google تجريبي يحفظ جلسة محلية، ومكانه جاهز لاستبداله بتهيئة Firebase Google.
- إنشاء مجلس عام/خاص.
- دخول بكود مجلس أو رابط دعوة `/invite/CODE`.
- طاولة لعب Mobile-first بتصميم مجلس عربي داكن ولمسات ذهبية.
- 4 أو 6 مقاعد ثابتة، مع مقاعد متاحة قبل بدء المجلس.
- اللاعب الحالي يظهر أسفل الطاولة.
- لوحة قرار ثابتة أسفل الشاشة تتغير حسب المرحلة.
- مرحلة انتظار، مزايدة، كشف نتيجة، وبين الجولات.
- سلم مزايدة يظهر فقط أثناء المزايدة.
- أزرار مزايدة: انسحب، مساومة، رأس المال كامل.
- يد اللاعب تظهر فقط بعد بدء الجولة.
- قائمة جانبية للدردشة، الانتظار، الترتيب، السجل، المشاهدين، الإدارة، والتقرير.
- بوتات مبدئية بمستويات سهل/متوسط/قوي.
- سجل أحداث وتقارير جولات وإحصائيات لاعب.
- PWA مع Service Worker غير عدواني وإصدار كاش واضح.

## ملاحظات مهمة

- لا يوجد دفع حقيقي ولا مراهنات مالية، كل الأرصدة نقاط افتراضية.
- Google Login الحالي واجهة/جلسة تجريبية حتى يتم ربط Firebase config الحقيقي.
- OTP السعودي مؤجل عمدًا حتى تستقر الطاولة وGoogle Login، التزامًا بالمواصفات.
- البيانات في الذاكرة حاليًا، ويجب ربط قاعدة بيانات لاحقًا قبل الإنتاج الحقيقي.
